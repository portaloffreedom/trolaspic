Using trolaspic


L'utilizzo di Trolaspic è molto semplice.

Carica mappe scritte in un formato speciale.map con l'immagine relativa.
Clicca sul nodo dove vuoi partire e sul nodo dove vuoi arrivare, premendo calcola il risultato verrà visualizzato a schermo.
